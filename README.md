## 블록체인 기반 정자 거래소, Sperman
<div align="center">
<br>

### 📑 App Introduction

  Superman은 허위 개인정보와 개인정보 훼손 없이 안전한 정자 거래를 도와주는 서비스입니다.


**Feature 1**  
- zkpass를 이용해 개인정보를 소유하지 않고 표기 가능합니다.

**Feature 2**
- 블록체인을 이용해 개인 의료 정보를 안전하게 가져옵니다.
